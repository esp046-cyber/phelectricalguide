# phelectricalguide
comprehensive Electrical Installation & Maintenance guide app for the Philippines with:
Key Features:
6 Main Sections: Basics, Installation, Maintenance, Safety, Tools, and Career paths
PEC Compliant: Based on Philippine Electrical Code standards
Practical Content: Step-by-step installation guides, troubleshooting tables, wire sizing charts
Safety Focus: Critical safety procedures, PPE requirements, first aid for electrical shock
Career Guide: How to become an electrician in PH, TESDA certification, salary ranges
Search Function: Find information quickly across all sections
Mobile-Optimized: 1% margins, small readable fonts, smooth scrolling
Design Elements:
Professional electrical theme (dark blues with red/yellow accents)
Color-coded cards (warnings in red, info in cyan)
Numbered step-by-step guides
Comprehensive reference tables
Clean, technical layout appropriate for a professional guide
The app covers everything from basic electrical concepts to advanced career guidance, all tailored for the Philippine electrical industry and 2026 standards!
